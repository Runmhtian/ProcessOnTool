# ProcessOnTool

利用processOn 支持倒入pos文件的功能，实现 文件的下载和倒入 突破文件数限制

## 相关配置
user_name 用户名

password 密码

self_file_max_num  processOn文件数限制 

self_pos_path  保持路径

## 相关命令

list local 查看本地文件

list online 查询线上文件

download all 下载所有线上文件到本地 并删除

import {title}  导入到线上   title 是 list local中的文件名

import path   导入文件到线上
